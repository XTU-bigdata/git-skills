## 安装Git

windows安装程序可以从下面的网页下载： http://git-scm.com/downloads

针对 OS X 的安装。

```shell
brew install git
```

 针对 Ubuntu 和使用 apt 的 Linux 发行版的安装。

```shell
apt-get install git 
```

针对 Fedora 和使用 yum 的 Linux 发行版的安装。 

```shell
yum install git
```



## 更新Git

```shell
# windows下
git --version
git update-git-for-windows
```



## 在命令行上访问Git帮助 

```shell
git help 
```

查看所有话题的列表 

```shell
git help --all 
```

查看 Git 的术语词汇表

```shell
git help glossary
```
